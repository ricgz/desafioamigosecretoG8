# DesafioamigosecretoG8

  Desafio de amigo secreto, es un juego web escrito en javascript, el cual permite
de forma acertiva realizar el sorteo del clasico juego **"amigo secreto"**.

## Caracteristicas:

- No admite nombres en blanco.
- Listado de nombres ingresados dinamicamente actualizable.
- Se requiere un minimo de 3 nombres para mantener el misterio.
- Cero posibilidad de sortear a la misma persona (registro interno de sorteados).

[Demo del juego](https://pages.github.com/).

  Para jugar se deben ingresar todos los nombres a sortear en la apliacion, a medida que se van  ingresando
estos seran agregados a un listado visible para todos, lo cual permitira confirmar que se han ingresado 
la totaliad de los nombres.

  Con motivo de mantener el misterio, se debe ingresar un minimo de 3 nombres para poder 
hacer iniciar el sorteo y asignacion del amigo secreto. Una vez desee iniciar el sorteo, solo debe presionar el boton **"Sortear Amigo"**, la lista se ocultara y comenzara el sorteo. Repetir 
este proceso hasta sortear la totalidad de los nombres.
